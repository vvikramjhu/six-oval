------------------------------------------------------------
SIX OVAL
------------------------------------------------------------

SIX OVAL is a Java class library for Open Vulnerability and 
Assessment Language (OVAL).

More information is available at 
http://staff.aist.go.jp/nakamura-akihito/six/


OVAL is sponsored by US-CERT at the U.S. Department of
